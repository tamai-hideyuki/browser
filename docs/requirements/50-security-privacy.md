# 50. セキュリティ / プライバシー 要件

## 脅威モデル
| 脅威 | 対策 |
|---|---|
| 悪意ある Web ページがプロセス侵害 | Chromium のサンドボックス + Site Isolation |
| 悪意ある Web ページがブラウザ機能悪用 | IPC ホワイトリスト・Origin 検証 |
| トラッカー・フィンガープリンティング | トラッカーブロック・FP 緩和 |
| 中間者攻撃 | HTTPS-only モード・HSTS |
| 悪意ある Boost / 拡張 | 権限明示・サンドボックス・Isolated World |
| ローカルデータ盗難 | OS ディスク暗号化に依存 / Keychain 利用 |
| 同期データ漏洩 | E2EE |

## サンドボックス
- Chromium のサンドボックスをそのまま利用
- レンダラプロセスは ファイルシステム・OS API に直接アクセス不可
- 必要な操作はメインプロセスへ IPC 経由で要求

## サイト分離（Site Isolation）
- デフォルトで有効
- 異なるオリジンは別プロセス
- クロスオリジン iframe も別プロセス
- 無効化オプションは提供しない

## HTTPS-only モード
- デフォルト：警告のみ（HTTP は警告バナーで通知）
- 設定で「HTTPS 強制」へ切替可能（HTTP アクセス時はブロック画面）
- HSTS Preload リスト準拠
- 証明書エラー：詳細表示 + 「危険を承知で進む」（オプション無効化可）

## トラッカーブロック
### 採用方針
- フィルタリストベース（EasyList / EasyPrivacy）
- Brave / uBlock の代表的フィルタを内蔵
- ユーザーがフィルタ追加可能

### ブロックレベル
1. **Standard**（デフォルト）：既知トラッカーをブロック
2. **Strict**：3rd party Cookie 全ブロック + フィンガープリント緩和
3. **Off**：すべて許可（サイト個別に許可も可能）

### サイトごとの上書き
- アドレスバー左の盾アイコンから ON/OFF
- 「このサイトでは許可」リスト

## フィンガープリンティング緩和
- Canvas / WebGL のノイズ化（Strict モードのみ）
- User-Agent の Reduced UA 採用（Chromium 標準）
- Audio API / フォントリスト撹乱（Strict モードのみ）
- ネイティブ実装はせず、Chromium 提供機能を最大活用

## 3rd Party Cookie
- フェーズ1：デフォルトで Strict（3rd party Cookie ブロック）
- 既知ブラウザ業界の流れに追従
- サイト個別の例外設定可

## プライベートウィンドウ
- `Cmd+Shift+N` で起動
- ウィンドウ独自の一時 Session（プロファイルから完全分離）
- 履歴 / Cookie / LocalStorage / Cache を一切残さない
- DNS over HTTPS 強制
- ウィンドウを閉じると全データ破棄

## DNS over HTTPS / HTTPS DNS
- デフォルト：システム設定に従う
- 設定で {Cloudflare 1.1.1.1, Google 8.8.8.8, NextDNS, カスタム} 選択可
- プライベートウィンドウは強制 ON

## CSP / セキュリティヘッダ
- シェル UI 自体に厳格な CSP を適用（`default-src 'self'`）
- `eval` / インラインスクリプト禁止（React の hydration を除く）
- IPC 経路はメッセージタイプの allowlist で検証

## 権限管理
Web ページからの権限リクエスト UI：
- カメラ / マイク / 位置情報 / 通知 / クリップボード読み取り / MIDI / USB / Bluetooth
- 各リクエストはアドレスバー下にバナー表示
- 「許可」「ブロック」「今回のみ」
- サイトごとの許可リストを `about:settings/permissions` で管理

## マルウェア / フィッシング保護
- フェーズ1：簡易（既知悪性URL の静的リスト）
- フェーズ2：Google Safe Browsing API 連携検討（プライバシーとのトレードオフ評価）
- 警告画面：オーバーレイ表示、「危険を承知で進む」リンク

## クリップボード保護
- ページからの自動 paste 読み取りはブロック
- ユーザーアクション起点のみ許可（標準仕様）

## アップデート署名
- 自動更新パッケージは Ed25519 署名検証
- 詳細は `90-update-telemetry.md`

## 監査ログ
- セキュリティイベント（証明書エラー / 権限付与 / トラッカーブロック数）を記録
- 設定 → プライバシーで集計表示
- ローカル保存のみ、外部送信なし

## 非機能要件
- トラッカーブロックによるページ表示遅延：< 5%
- Strict モードの追加コスト：< 50ms / ページ

## スコープ外
- VPN 内蔵（フェーズ4 検討）
- Tor 接続
- マルウェアスキャナの内蔵
