# 00. プロジェクト概要

## ビジョン
Arcブラウザに比肩する「ワークスペース型ブラウザ」を構築する。タブを「開きっぱなしのページ」ではなく「整理されたワークスペース」として扱い、サイドバー・Spaces・コマンドバーを核としたUX体験を提供する。

## ターゲットユーザー
- 同時に大量のタブ・プロジェクトを扱う開発者・リサーチャー・ナレッジワーカー
- 既存ブラウザのタブ管理に不満を持つヘビーユーザー
- macOS / Windows のデスクトップユーザー（フェーズ1ではmacOS優先）

## プロダクトの差別化要素（Arcに準拠）
1. **サイドバー中心UI**：上部タブバーではなく左サイドバーにタブ・お気に入り・Spacesを集約
2. **Spaces**：用途ごと（仕事・プライベート・学習等）にタブ群とプロファイルを切り替え
3. **コマンドバー**：`Cmd+T` で URL入力・検索・タブ移動・アクションを統合
4. **Boosts**：任意のサイトに CSS / JS を注入してパーソナライズ
5. **Little Arc**：他アプリから開いたリンクを軽量ポップアップで表示
6. **AI機能**：ページ要約・Ask on Page・タブ整理支援
7. **タブの自動アーカイブ**：開きっぱなしの放置タブを自動でアーカイブ

## スコープ（フェーズ1 = MVP）
- macOS デスクトップアプリとして動作
- Chromium ベース WebView を埋め込み、Web 互換性は Chromium に委譲
- サイドバー・Spaces・タブ管理・コマンドバーの基本UX
- 単一プロファイル + ローカル永続化（同期は後フェーズ）

## スコープ外（フェーズ1 では作らない）
- 独自レンダリングエンジン（Chromium 埋め込みで代替）
- モバイル版（iOS / Android）
- マルチデバイス同期 / クラウドアカウント
- 完全な Chrome 拡張互換
- Easel / Notes 等のコラボ機能

## 非目標（プロジェクト全体としても作らない）
- Chromium / WebKit / Gecko に並ぶ独自レンダラ
- 既存ブラウザの完全なクローン（独自UXを優先）
- エンタープライズ管理機能（MDM / GPO 連携）

## 用語集
| 用語 | 意味 |
|---|---|
| Space | タブ群とテーマ・プロファイルをまとめた作業空間 |
| Pinned Tab | サイドバー上部に常駐し、閉じても残るタブ |
| Today Tab | 一時的に開いているタブ。一定期間で自動アーカイブ |
| Boost | 特定サイトに注入する CSS / JS のセット |
| Little Arc | 外部リンクを開く軽量ポップアップウィンドウ |
| Command Bar | `Cmd+T` で起動する統合コマンド入力UI |
| WebView | 埋め込み Chromium レンダラのインスタンス |

## 成功指標
- MVP 段階で日常ブラウジング（検索・SNS・ドキュメント閲覧・動画再生）が可能
- 100 タブ同時保持時に常用に耐えるメモリ・起動速度
- ユーザー自身が「メインブラウザ」として置き換えられる完成度
