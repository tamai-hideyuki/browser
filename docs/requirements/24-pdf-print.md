# 24. PDF / 印刷 要件

## PDF 閲覧
### 採用
Chromium 内蔵の PDFium ベース PDF ビューアを利用する。独自実装はしない。

### 機能
- PDF を URL でロード時に自動的にビューアで表示
- ローカルファイル（`file://`）の PDF も表示可
- ドラッグ&ドロップでブラウザに PDF を投入 → 新規タブで開く

### ビューア UI
Chromium 標準 UI を採用しつつ、テーマ統合は M2 以降：
- ツールバー：ズーム / ページ送り / フィット / 回転 / ダウンロード / 印刷
- サイドバー：サムネイル / アウトライン / 添付ファイル
- 検索：`Cmd+F`
- フォーム入力対応
- 注釈表示（編集は不可）

### ナビゲーション
- ページジャンプ：`Cmd+Option+G`（標準的なページ番号入力）
- 次ページ：`PageDown` / `→`
- 前ページ：`PageUp` / `←`

## 印刷
### 起動
- `Cmd+P` で印刷ダイアログ
- ページ右クリック → 印刷
- PDF ビューアの印刷ボタン

### 印刷プレビュー
- ダイアログ内に印刷プレビュー表示
- 余白・向き・拡大率の即時プレビュー

### 印刷オプション
| 項目 | 内容 |
|---|---|
| プリンタ | OS にインストール済みのプリンタ一覧 + PDF として保存 |
| ページ範囲 | 全ページ / 現在のページ / カスタム範囲 |
| 部数 | 1〜999 |
| カラー / グレースケール | |
| 用紙サイズ | A4 / Letter / Legal / カスタム |
| 向き | 縦 / 横 |
| 余白 | デフォルト / なし / 最小 / カスタム |
| 拡大率 | デフォルト / カスタム % |
| ページ毎の枚数 | 1 / 2 / 4 / 6 / 9 / 16 |
| 両面 | 片面 / 長辺綴じ / 短辺綴じ（プリンタ依存） |
| ヘッダー・フッター | ON / OFF |
| 背景グラフィック | ON / OFF |

### ヘッダー・フッターの内容
- 左上：ページタイトル
- 右上：ページ番号 / 総ページ数
- 左下：URL
- 右下：日時

カスタマイズは M2 以降。

### PDF として保存
- プリンタ選択で「PDF として保存」を選択
- ファイル選択ダイアログで保存先を指定
- macOS は OS のプレビューサービスとは別経路（Chromium 内蔵）

### Web ページ → PDF
独自機能：「ページを PDF として保存」コマンド
- メニュー：ファイル → ページを PDF として保存
- コマンド：`/save-as-pdf`
- リーダーモード版 PDF（M2 以降）

### 印刷設定の保存
- 直近の設定を記憶（プリンタ・余白・向き等）
- サイトごとには記憶しない（複雑性回避）

## ダウンロード扱い
PDF を「常にダウンロード」する設定：
- 設定 → ダウンロード → 「PDF を開かずに保存」トグル
- ON 時：Content-Type が PDF のレスポンスはダウンロードキューへ

## 印刷の実装
Electron の `webContents.print()` API を使用：
```typescript
view.webContents.print({
  silent: false,
  printBackground: true,
  // ... ユーザー設定
}, (success, failureReason) => {
  // ...
});
```

ダイアログ自体は OS 標準 / Chromium 標準を使用。独自ダイアログは作らない。

## エラーハンドリング
- PDF 読み込み失敗：エラーオーバーレイ表示（[08-navigation.md](../design/08-navigation.md) と同様）
- 印刷エラー：トースト通知 + 詳細ログ
- プリンタ未検出：「プリンタが見つかりません」案内

## キーボードショートカット
| ショートカット | 機能 |
|---|---|
| `Cmd+P` | 印刷 |
| `Cmd+Shift+P` | 印刷ダイアログを「PDF として保存」モードで開く（M2） |

## 非機能要件
- 100MB の PDF を開くまで 5 秒以内
- 印刷ダイアログ起動 1 秒以内
- 印刷プレビュー更新 500ms 以内

## アクセシビリティ
- スクリーンリーダーで PDF テキストが読み上げ可能（Chromium 標準）
- 印刷ダイアログはキーボード操作のみで完結

## スコープ外
- 独自 PDF 編集機能
- PDF 注釈の追加・編集
- OCR
- フォーム自動入力（Chromium 標準を超える機能）
